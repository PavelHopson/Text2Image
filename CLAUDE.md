# text2image-studio

## Tech Stack

- **Primary:** TypeScript + React + Vite
- **Package Manager:** npm

## Commands

```bash
# Install
npm install

# Dev
npm run dev

# Build
npm run build

```

LICENSE/

## Code Style

- Use TypeScript strict mode when available
- Prefer functional components and hooks in React
- Use named exports over default exports
- Keep files under 300 lines
